# Portfolio OS

This project is a personal portfolio designed to look and feel like a desktop operating system. It's built with React and Vite, and it features a window-based interface with a dock, draggable windows, and a variety of applications to showcase my work and skills.

## Features

*   **Desktop-like UI:** A complete desktop experience with a dock, menu bar, and draggable and resizable windows.
*   **Dynamic Windows:** Open multiple applications in their own windows, move them around, and resize them.
*   **A Collection of Apps:**
    *   **Finder:** Browse through projects and files.
    *   **Safari:** A simulated web browser to view live projects.
    *   **Terminal:** A fake terminal that displays my skills and experience.
    *   **Photos:** A gallery to showcase images.
    *   **TextEdit:** A simple text editor to view my resume.
    *   **Contact:** A form to get in touch with me.
*   **Responsive Design:** The portfolio is designed to work on different screen sizes.

## Technologies Used

*   **React:** The core library for building the user interface.
*   **Vite:** A fast build tool and development server.
*   **Zustand:** A small, fast, and scalable state management solution.
*   **GSAP:** For animations and interactive elements.
*   **Tailwind CSS:** For styling the application.
*   **Lucide React:** A library of beautiful and consistent icons.

